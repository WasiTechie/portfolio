Photo Booth

Photo Booth App

Sept 19 - Oct 5, 2017/Computer Science/Engineering

I used Massachusetts Institute of Technology's App Inventor to create the "Photo Booth” app. The Photo Booth app was my second project. There was a tutorial given for certain parts of the app, but all of the extra features had to be coded. I worked with Vince Tan, and together we were challenged to add more ideas and useful features to the application. Scratch is very similar to App Inventor, though App Inventor allows for more flexibility and features. Also, App Inventor is meant to create Android Apps, whereas Scratch can only create apps for the Scratch platform. After a few weeks of tough coding and debugging, Vince and I were able lock in and create an amazing and useful application that allows the user to take a picture using the device's front and back facing camera and edit the picture using lines and colors. After taking the picture from the camera page, the picture is stored on one of the four canvases. The picture can be taken to the edit page from there. The user then has the flexibility to draw on the image, and apply different effects to it. Overall this was a successful project as we were able to implement everything we wanted into the application. During this project, I learned that even the smallest typos can make a huge difference. Also, debugging is a crucial and tough part in the programming process.
